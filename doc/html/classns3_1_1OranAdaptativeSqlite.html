Detailed Description

The OranAdaptativeSqlite class is a new implementation of the database for the Near-Rt RIC.

This database is accepts custom reports that inherit from OranReportSqlite. The tables of this database are created dinamically based on the report information. Therefore, the user can define reports with arbitrary information.
