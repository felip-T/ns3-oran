Detailed Description

Logic Module for the Near-RT RIC that issues Commands to handover from an LTE cell to another based on an ONNX ML model.

◆ Run()

std::vector< Ptr< OranCommand > > ns3::OranLmLte2LteOnnxHandover::Run ( void  )
overridevirtual

Runs the logic specific for this Logic Module. This will retrieve the location of all LTE UEs and eNBs and the application loss for all UEs, calculate the distance between all eNBs for each UE, pass these as inputs to the ONNX ML model, and then generate zero or more handover Commands based on the ML model output that is generated.

Returns
A vector with the handover commands generated by this Logic Module.
