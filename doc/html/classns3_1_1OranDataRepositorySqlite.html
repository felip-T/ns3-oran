Detailed Description

A Data Repository implementation that uses an SQLite database as the storage backend.

The database used as the backend may be created new, or it may be an existing database. This class does not provide methods for deleting existing database files; if this is required, the user must take care of that in the scenario.

The methods defined in the OranDataRepository API build SQL prepared statements to access the database, validating the return code after each database query.
