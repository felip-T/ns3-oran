\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classns3_1_1OranReportSqlite}{Oran\+Report\+Sqlite}} class is a base class for all custom reports to be used with the \mbox{\hyperlink{classns3_1_1OranAdaptativeSqlite}{Oran\+Adaptative\+Sqlite}} database. 

It is expected that the user\textquotesingle{}s custom report inherits from this class and implements the Get\+Table\+Info and Get\+Table\+Name virtual methods, as well as the To\+String method, inherited from \mbox{\hyperlink{classns3_1_1OranReport}{Oran\+Report}} base class.
