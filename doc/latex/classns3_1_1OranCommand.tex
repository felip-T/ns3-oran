\doxysubsection{Detailed Description}
Base class of a Command. This class provides the implementation of the Get\+Target\+E2\+Node\+Id method for all the Commands. This base class can be instantiated, but the Command issued lacks any meaning and will be silently ignored by the E2 Node Terminators.
