\doxysubsection{Detailed Description}
A Command instructing an LTE e\+NB to handover a UE to another LTE e\+NB. In this command the Target E2 Node ID is the serving LTE e\+NB, and the command contains the cell ID of the e\+NB to handover to, and the RNTI of the UE to be handoverd.
