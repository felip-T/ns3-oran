\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classns3_1_1OranAdaptativeSqlite}{Oran\+Adaptative\+Sqlite}} class is a new implementation of the database for the Near-\/\+Rt RIC. 

This database is accepts custom reports that inherit from \mbox{\hyperlink{classns3_1_1OranReportSqlite}{Oran\+Report\+Sqlite}}. The tables of this database are created dinamically based on the report information. Therefore, the user can define reports with arbitrary information.
