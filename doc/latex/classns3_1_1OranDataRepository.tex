\doxysubsection{Detailed Description}
Base class for the Data Repository module. This class defines the API for accessing the data stored by the RIC, regardless of the actual implementation. This allows other models to access the data without having to know the details of the storage backend.

When the Data Repository is deactivated all the requests to store data will be ignored, and all queries for stored data will return empty sets.
