\doxysubsection{Detailed Description}
Base class for the Logic Modules that make up the logic of the Near-\/\+RT RIC. This base class provides implementations for common operations, like activation, deactivation, getters and setters, and logging logic traces to the Data Repository.

This class cannot be instantiated as it lacks implementation of the Run method.
